**MobiStock v2.0** is a modern, lightweight, and professional inventory management system specifically designed for mobile phone retailers. It features a sleek "Enterprise Dark" interface with real-time business intelligence and transaction logging.

---

## 🚀 Key Features

### 📊 Business Intelligence Dashboard
*   **Real-time Analytics:** Track Total Revenue, Net Profit, and Inventory Value at a glance.
*   **Margin Analysis:** Automatically calculates your average profit margin percentage.
*   **Visual Data:** Interactive charts showing stock distribution by brand using Matplotlib.

### 📦 Smart Inventory Management
*   **Advanced Filtering:** Search by model or brand instantly.
*   **Critical Stock Alerts:** Dedicated "Low Stock" filter and visual red highlights for items that need restocking.
*   **Profit Tracking:** Detailed view of purchase vs. sale prices and individual product margins.

### 📜 Transaction History (Audit Log)
*   **Full Transparency:** Every sale and restock is logged with a precise timestamp.
*   **Cash Flow Tracking:** See exact money flow (Green for revenue, Red for expenses) in a dedicated history terminal.

### 🔄 Transaction Terminal
*   **Quick Actions:** Simple "Restock" and "Sell" interface.
*   **Validation:** Smart logic prevents selling items that are out of stock.

## 🎨 Enterprise Design
The UI is built with a **Deep Dark Theme** focus:
- 🌑 **Background:** `#121212` (Modern AMOLED-friendly dark)
- 🔵 **Primary:** `#3B8ED0` (Professional Blue)
- 🟢 **Success:** `#2FA572` (Growth Green)
- 🔴 **Danger:** `#D3514D` (Alert Red)
